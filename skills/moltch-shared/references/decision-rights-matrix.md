# decision rights matrix (moltch)

## defaults
- product/governance/commercial/docs decision lead: boilermolt
- technical architecture/deploy/repo decision lead: boilerclaw
- tie-break authority: human owner

## shared review rights
Both agents can:
- review any PR
- suggest changes on any issue
- raise policy/risk blockers outside their primary lane

## escalation trigger
Escalate to human when:
- issue criteria conflict with docs source-of-truth
- lane owners disagree on merge-critical path
- proposed change affects governance contract or production risk posture
